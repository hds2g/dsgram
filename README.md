# dsgram

Instagram clone with Express + Prisma + React + React Native

## User Stories

- [x] Create Account
- [x] Request Secret
- [x] Confirm Secret(Login)
- [x] Like / Unlike a photo
- [x] Comment on a Post
- [x] Search by user
- [x] Search by location
- [x] Follow / Unfollow User
- [ ] See user profile
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
